**Release Date**: April 30, 2024

## 🎉 What's New

### Major Features

#### 1. **Dynamic Theme System**
- 6 promotion-based themes (WWE, AEW, NJPW, TNA, ROH, CMLL)
- Real-time theme switching
- Color customization per promotion
- Persistent theme selection

#### 2. **Advanced Animations**
- Framer Motion integration
- 8 types of smooth animations
- GPU-accelerated 60 FPS performance
- Micro-interactions for UI feedback

#### 3. **Modern UI Components**
- Redesigned wrestler card system
- Match card visualization
- Data visualization with Chart.js
- In-universe notification system
- Simulation feedback modal

#### 4. **Performance Optimizations**
- LRU cache system (350 entry limit)
- Function memoization
- Debouncing and throttling
- Optimized data processing

### Minor Features

- Improved error handling
- Better loading states
- Enhanced responsive design
- Accessibility improvements

## 📦 Installation

### Windows

#### Option 1: Installer
1. Download `Pro-Wrestling-Sim-3.0.0.exe`
2. Run the installer
3. Follow on-screen instructions
4. Launch from Start Menu

#### Option 2: Portable
1. Download `Pro-Wrestling-Sim-3.0.0-portable.zip`
2. Extract to desired location
3. Run `Pro Wrestling Sim.exe`
4. No installation required

### System Requirements

**Minimum**:
- Windows 7 SP1 or later
- 2 GB RAM
- 500 MB disk space
- Intel Core i3 or equivalent

**Recommended**:
- Windows 10 or later
- 4 GB RAM
- 1 GB SSD
- Intel Core i5 or equivalent

## 🔄 Update Instructions

### From v2.x to v3.0.0

1. **Backup Your Data**
   ```
   Copy your save files to a safe location
   ```

2. **Uninstall Previous Version**
   ```
   Control Panel > Programs > Uninstall a program
   Select Pro Wrestling Sim
   Click Uninstall
   ```

3. **Install v3.0.0**
   ```
   Download Pro-Wrestling-Sim-3.0.0.exe
   Run installer
   Follow on-screen instructions
   ```

4. **Restore Your Data**
   ```
   Copy save files back to new installation
   ```
