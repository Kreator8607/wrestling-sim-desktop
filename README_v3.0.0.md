## 📋 Visão Geral

Pro Wrestling Sim Desktop é uma aplicação desktop completa para simular eventos de wrestling profissional. Com mais de 5,000 lutadores, 100+ promoções e 100+ títulos, você pode criar e gerenciar sua própria história de wrestling.

**Versão:** 3.0.0 (Lançamento Completo)  
**Tamanho:** 165 MB  
**Requisitos:** Windows 7 ou superior (x64)

## 🚀 Instalação

### Requisitos
- Windows 7 ou superior (64-bit)
- 200 MB de espaço em disco
- 2 GB de RAM (recomendado 4 GB)

### Opção 1: Instalador NSIS (Recomendado)
1. Baixe `Pro Wrestling Sim v3.0.0.exe` do [GitHub Releases](https://github.com/Kreator8607/wrestling-sim-desktop/releases)
2. Execute o instalador
3. Siga as instruções na tela
4. Clique em "Concluir"

### Opção 2: Executável Portátil
1. Baixe `Pro Wrestling Sim v3.0.0 Portable.exe`
2. Execute diretamente (sem instalação)
3. Nenhuma configuração necessária

### Opção 3: Compilar do Código-Fonte
```bash
# Clone o repositório
git clone https://github.com/Kreator8607/wrestling-sim-desktop.git
cd wrestling-sim-desktop

# Instale as dependências
npm install

# Compile para Windows
npm run build:win
```

## 🎮 Como Usar

### Primeira Vez
1. Abra a aplicação
2. Siga o tutorial interativo (8 passos)
3. Crie seu primeiro evento
4. Simule matches e acompanhe rankings

### Criar um Evento
1. Clique em "Booking" no menu lateral
2. Clique em "Novo Evento"
3. Selecione lutadores e tipos de match
4. Configure objetivos (opcional)
5. Simule o evento

### Acompanhar Rankings
1. Clique em "Rankings" no menu lateral
2. Veja o ranking dos lutadores
3. Filtre por promoção ou atributo
4. Acompanhe progresso em tempo real

### Gerenciar Títulos
1. Clique em "Títulos" no menu lateral
2. Selecione uma promoção
3. Veja campeões atuais
4. Acompanhe histórico de reinados

### Modo Carreira
1. Clique em "Carreira" no menu lateral
2. Selecione um lutador
3. Veja estatísticas de carreira
4. Complete objetivos pessoais

### Achievements
1. Clique em "Achievements" no menu lateral
2. Veja todos os achievements disponíveis
3. Acompanhe progresso
4. Ganhe pontos ao desbloquear

## 💾 Backup e Recuperação

### Criar Backup Manual
1. Clique em "Backup" no menu lateral
2. Clique em "Criar Backup Agora"
3. Backup será salvo automaticamente

### Auto-Backup
1. Vá para "Backup"
2. Ative "Auto-backup automático"
3. Configure intervalo (padrão: 5 minutos)

### Restaurar Backup
1. Vá para "Backup"
2. Clique em "Restaurar" no backup desejado
3. Confirme a ação
4. Aplicação será recarregada

### Exportar Backup
1. Vá para "Backup"
2. Clique em "Exportar" no backup desejado
3. Arquivo JSON será baixado

## 🐛 Troubleshooting

### Problema: Aplicação não inicia
**Solução:**
1. Verifique se tem Windows 7 ou superior
2. Instale o Visual C++ Redistributable
3. Reinicie o computador
4. Tente reinstalar

### Problema: Dados não salvam
**Solução:**
1. Verifique espaço em disco
2. Verifique permissões de pasta
3. Tente criar backup manual
4. Restaure de backup anterior

### Problema: Performance lenta
**Solução:**
1. Limpe cache (Configurações → Reset)
2. Reduza número de eventos
3. Feche outras aplicações
4. Aumente RAM disponível

### Problema: Erro ao importar dados
**Solução:**
1. Verifique formato do arquivo (JSON/CSV)
2. Verifique tamanho do arquivo (<10MB)
3. Valide estrutura do arquivo
4. Tente exportar e reimportar
